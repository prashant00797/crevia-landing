# Crevia Landing Page

A modern landing page for **Crevia** — an AI-powered creator operating system designed to help creators manage content, track workflows and generate AI-powered content from a single workspace.

## Features

- Product-focused hero section
- Interactive product preview
- Feature highlights
- Launch announcement modal
- Fully responsive design

## Tech Stack

- Next.js (Ts)
- Tailwind CSS
- Shadcn
